College Football Database Website

website link: cfdb.me
github: College_Football_Database
github wiki: project implementation information

Description: cfdb.me is a College Football Database Website created by Nick Jordan, Cameron Reese, David Sipos, Joshua Waller, and Chris LeSage as part of the Software Engineering course at The University of Texas, Summer 2015. The website showcases every conference, team, and player in NCAA College Football Divison I FBS in the 2014 season, including stats, schedules, and player information. The database contains over 14,000 entries.

I was primarily responsible for the layout, design, and graphical user interface/front-end portions of the website. I initialized the GUI using a premade Bootstrap theme, but soon veered far from the original template code. I created the GUI template based on my original conceptual layout and design of the website, and with the help from other team mates, I populated the pages with the appropriate data. I worked with HTML, CSS, Javascript, and Bootstrap during this projects.

Netflix Movie Prediction Algorithm

github: Netflix_Movie_Predictions

Description: Here, my partner and I developed an algorithm for the Netflix Prize coding competition held in 2009. The prize would award $1 Million to the group who could develop an algorithm to predict user movie ratings at least 10% better than the one they already had. We wrote our algorithm in 2015, so obviously the compeition had since ended. We were only given 3 types of data points: a list of every user's previous movie ratings, and a list of all ratings for each movie, and the movie release date of each movie. The actual competitors were allowed to integrate essentially endless amounts of data into their algorithms, but despite our relatively specific data set, we still managed to develop an algorithm that could predict user movie ratings 2% better than Netflix had done prior to the competition. Netflix's RMSE value from their algorithm was 0.95, where ours was 0.93. Although it is not remarkably better, I believe there is still some merit to the fact that we beat Netflix, no matter by how much or little.
